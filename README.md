# Portfólio Profissional - Diogo Chaves Torres

Este projeto consiste no desenvolvimento de um website de portfólio profissional para apresentar minha trajetória, projetos, experiências e formas de contato.

O site foi desenvolvido como atividade da disciplina **Laboratório de Desenvolvimento de Software** do curso de **Engenharia de Software**.

---

# Funcionalidades

O website possui as seguintes seções:

### Sobre Mim
Apresentação pessoal em português e inglês, destacando formação, objetivos profissionais e interesses na área de Tecnologia da Informação.

### Projetos
Lista de projetos desenvolvidos com:

- Nome do projeto
- Descrição
- Tecnologias utilizadas
- Link para o repositório no GitHub
- Imagens do projeto

### Experiências
Experiências profissionais com:

- Nome da empresa
- Cargo
- Período
- Descrição das atividades

### Contato
Seção com:

- Ícones clicáveis para Email, LinkedIn e WhatsApp
- Formulário de envio de mensagem
